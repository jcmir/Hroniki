# Backup & Restore — Cross-Device Test

**Инвариант HRONIKI**: «Мои воспоминания принадлежат мне.»

Этот тест является **Beta Definition of Done**.
Резервная копия — единственная защита от потери данных при смене устройства или переустановке.

---

## Подготовка

На обоих устройствах установлена одна и та же версия APK:
```
app.hroniki.mobile v0.2.1-beta (versionCode 21)
```

## Граничные случаи (опционально)

| # | Сценарий | Ожидаемое |
|---|---|---|
| E1 | Неверный пароль при импорте | Ошибка расшифровки, данные не испорчены |
| E2 | Повреждённый файл бэкапа | Ошибка, приложение продолжает работать |
| E3 | Импорт в непустую БД | Данные объединяются или предупреждение |
