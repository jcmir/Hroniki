# Smoke Flow Script — HRONIKI Beta 0.2.1

Пошаговый тест-сценарий для устройства. Выполнять строго последовательно.
Версия APK: `app.hroniki.mobile` / `versionCode 21` / `0.2.1-beta`

---

## Подготовка

```bash
# Установка через adb (debug/release)
adb install -r build/outputs/apk/release/app-release.apk

# Очистка данных (если было предыдущая установка)
adb shell pm clear app.hroniki.mobile
```

## Definition of Done

Smoke Flow считается пройденным если:
- [ ] Все шаги 1–6 завершены без краша
- [ ] Записи восстанавливаются после process kill (шаг 5.3)
- [ ] PIN-экран блокирует доступ (шаг 6.6)
- [ ] Файл бэкапа создан (шаг 7.3)
