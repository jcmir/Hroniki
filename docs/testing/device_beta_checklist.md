# Android Device Beta Testing Checklist

Чек-лист проверки качества и сценариев тестирования бета-версии ХРОНИКИ на реальных Android смартфонах и эмуляторах (`app.hroniki.mobile`).

---

## 1. Установка и Пакетные Метаданные
- [ ] Идентификатор пакета установленного приложения: `app.hroniki.mobile`.
- [ ] Версия сборки в настройках: `Beta 0.2.1`.
- [ ] APK успешно устанавливается через `adb install hroniki.apk` без ошибок подписи.

## 2. Мастер Первого Запуска (FirstRun Wizard)
- [ ] Быстрый анимационный сплэш-экран `✦ ХРОНИКИ` отображается 300ms.
- [ ] Появляется окно выбора стартовых категорий и кнопка "Загрузить Демо-Данные".
- [ ] Выбор демо-данных наполняет Ленту и Сетку Объектов примерными хрониками.

## 3. Навигация и Экран «Центр Памяти» (`/reminders`)
- [ ] Навигационная панель содержит 4 активных таба (`Лента`, `Объекты`, `Напоминания`, `Настройки`).
- [ ] Нажатие на "Напоминания" открывает Центр Памяти с разделением по блокам (`Сегодня`, `Год назад`, `Будущие события`).

## 4. Бэкап и Безопасность
- [ ] Экспорт зашифрованного бэкапа сохраняет файл `.hroniki` с валидным `manifest.json`.
- [ ] Имитация перезапуска и ввод PIN-кода восстанавливают сохраненные данные.
