# Спецификация модуля: User Settings & Backup Manifest

Документ описывает систему пользовательских настроек, lifecycle первого запуска и спецификацию экспорта/импорта резервных копий.

---

## 1. Жизненный Цикл Первого Запуска (FirstRun State)

- `FirstRun`: Первичная инициализация при отсутствии сохраненного master key и PIN.
- `Setup`: Пошаговый мастер (ввод имени, установка PIN, выбор категорий).
- `Ready`: Готовность системы и переход к главному Таймлайну.

---

## 2. Структура Архива Резервной Копии (Backup Zip)

Состав экпортируемого архива:
- `manifest.json`: Версия схемы, название приложения и дата создания.
- `database.enc`: Зашифрованная база данных SQLite.
- `media.enc`: Архив прикрепленных фотографий и оригиналов.
