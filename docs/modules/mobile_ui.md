# Спецификация модуля: Mobile UI

Модуль `mobile_ui` включает в себя стили дизайна, Svelte 5 компоненты и инкапсулирует представление персонального дневника хроник.

---

## 1. Компоненты Пользовательского Интерфейса

- **`Timeline.svelte` / `+page.svelte`**: Главная страница ленты воспоминаний с группировкой по датам, поисковой строкой и кнопкой быстрой блокировки.
- **`EntryCard.svelte`**: Карточка хронологического события. Отображает дату, время, категорию, превью фотографий и текст.
- **`CreateEntryModal.svelte`**: Форма создания события с поддержкой добавления фото и реактивным сохранением.
- **`PINLockModal.svelte`**: Защитный оверлей аутентификации по PIN-коду. Открывается автоматически при переходе `session.ts` в состояние `awaitingUnlock`.

---

## 2. Дизайн-система (Apple Photos / Day One Style)

- **Палитра**:
  - Основной фон: `#0D0F12` (Deep Obsidian).
  - Карточки: `#161920` (Dark Slate) с тонкими границами `#282C37`.
  - Акценты: `#F59E0B` (Warm Amber) и `#3B82F6` (Muted Sapphire).
  - Текст: `#F3F4F6` (Crisp Soft White) и `#9CA3AF` (Muted Grey).
- **Типографика**: Чистый системный санс-сериф (`Inter`, `-apple-system`, `system-ui`).
- **Сетки и отступы**: Радиусы скругления 16px, комфортные отступы для сенсорных экранов.
